#### Boolean 对象属性

| 属性	             | 描述                     |
| ---                | ---                     |
| Boolean.length	 | length 属性，值为 1。 |
| Boolean.prototype	 | Boolean 构造函数的原型对象。 |
| Boolean.constructor| 返回对创建此对象的 Boolean 函数的引用 |

#### Boolean 对象方法

| 属性	             | 描述                     |
| ---                | ---                     |
| toString()	     | 把逻辑值转换为字符串，并返回结果。 |
| constructor        | 返回 Boolean 对象的原始值。 |
